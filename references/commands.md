# 常用 Bash 命令参考

## 文件与目录操作

### ls - 列出目录内容
| 选项 | 含义 |
|------|------|
| -l | 长格式，显示权限、所有者、大小、时间 |
| -a | 显示所有文件，包括隐藏文件（以 . 开头） |
| -h | 人类可读的大小格式（K, M, G） |
| -R | 递归列出子目录 |
| -t | 按修改时间排序（最新在前） |
| -r | 反向排序 |
| -S | 按文件大小排序 |

### cd - 切换目录
| 参数 | 含义 |
|------|------|
| 无参数或 ~ | 切换到主目录 |
| - | 切换到上一个工作目录 |
| .. | 切换到上级目录 |
| . | 当前目录（无变化） |
| / | 根目录 |
| 相对路径 | 相对于当前目录 |
| 绝对路径 | 从根目录开始的完整路径 |

### cp - 复制文件/目录
| 选项 | 含义 |
|------|------|
| -r 或 -R | 递归复制目录 |
| -p | 保留文件属性（权限、时间戳等） |
| -f | 强制覆盖，不提示 |
| -i | 交互式，覆盖前确认 |
| -v | 显示复制过程 |
| -u | 只复制更新的文件（源文件比目标新） |

### mv - 移动/重命名
| 选项 | 含义 |
|------|------|
| -i | 交互式，覆盖前确认 |
| -f | 强制覆盖，不提示 |
| -n | 不覆盖已存在的文件 |
| -v | 显示移动过程 |
| -u | 只移动更新的文件 |

**用法：**
- `mv old new` - 重命名文件/目录
- `mv file dir/` - 移动文件到目录
- `mv file1 file2 dir/` - 移动多个文件到目录

### rm - 删除文件/目录
| 选项 | 含义 |
|------|------|
| -r 或 -R | 递归删除目录及其内容 |
| -f | 强制删除，不提示确认 |
| -i | 交互式，删除前确认 |
| -v | 显示删除过程 |

⚠️ **注意**：rm -rf 是危险命令，删除后无法恢复！

### mkdir - 创建目录
| 选项 | 含义 |
|------|------|
| -p | 创建多级目录，存在也不报错 |
| -v | 显示创建过程 |
| -m MODE | 设置权限模式 |

**示例：**
- `mkdir dir` - 创建单个目录
- `mkdir -p a/b/c` - 创建多级目录
- `mkdir dir1 dir2 dir3` - 创建多个目录

### touch - 创建文件/更新时间戳
| 选项 | 含义 |
|------|------|
| -a | 只更新访问时间 |
| -m | 只更新修改时间 |
| -d STRING | 使用指定时间 |
| -c | 文件不存在时不创建 |

**用途：**
- 文件不存在时：创建空文件
- 文件存在时：更新访问和修改时间

### cat - 查看/连接文件
| 选项 | 含义 |
|------|------|
| -n | 显示行号 |
| -b | 显示行号（空行除外） |
| -s | 合并连续空行 |
| -A | 显示所有字符（包括换行符、制表符等） |

### find - 查找文件
| 选项 | 含义 |
|------|------|
| -name PATTERN | 按文件名匹配 |
| -iname PATTERN | 按文件名匹配（忽略大小写） |
| -type TYPE | 按类型：f=文件, d=目录, l=链接 |
| -size N | 按大小：+N=大于, -N=小于, N=等于 |
| -mtime N | 按修改时间：+N=超过N天, -N=N天内 |
| -exec CMD {} \; | 对每个匹配文件执行命令 |
| -exec CMD {} + | 批量执行（更高效） |
| -delete | 删除匹配的文件 |
| -maxdepth N | 最大搜索深度 |
| -mindepth N | 最小搜索深度 |

**示例：**
- `find . -name "*.txt"` - 查找所有 .txt 文件
- `find /tmp -type f -mtime +7` - 查找超过7天的文件
- `find . -size +100M` - 查找大于100MB的文件

## 文本处理

### grep - 文本搜索
| 选项 | 含义 |
|------|------|
| -i | 忽略大小写 |
| -v | 反向匹配（显示不匹配的行） |
| -r 或 -R | 递归搜索目录 |
| -n | 显示行号 |
| -c | 只显示匹配行数 |
| -l | 只显示包含匹配的文件名 |
| -L | 只显示不包含匹配的文件名 |
| -w | 匹配整个单词 |
| -E | 使用扩展正则表达式（等同于 egrep） |
| -F | 固定字符串，不解释正则（等同于 fgrep） |
| --color | 高亮显示匹配内容 |

**示例：**
- `grep "error" log.txt` - 在文件中搜索 error
- `grep -r "TODO" ./src` - 递归搜索目录
- `grep -E "error\|warning" log.txt` - 搜索多个模式

### sed - 流编辑器
| 命令 | 含义 |
|------|------|
| s/old/new/ | 替换第一个匹配 |
| s/old/new/g | 替换所有匹配（全局） |
| s/old/new/i | 忽略大小写替换 |
| -i | 就地修改文件 |
| -n | 抑制默认输出 |
| -e | 执行多个命令 |
| p | 打印匹配行 |
| d | 删除匹配行 |

**示例：**
- `sed 's/old/new/g' file.txt` - 替换并输出
- `sed -i 's/old/new/g' file.txt` - 就地替换
- `sed -n '10,20p' file.txt` - 打印第10-20行
- `sed '/pattern/d' file.txt` - 删除匹配行

### awk - 文本处理语言
| 变量 | 含义 |
|------|------|
| $0 | 整行内容 |
| $1, $2, ... | 第1、2...列（按分隔符分割） |
| NF | 当前行的字段数（列数） |
| NR | 当前行的行号 |
| FS | 输入字段分隔符（默认空格） |
| OFS | 输出字段分隔符 |
| RS | 输入记录分隔符（默认换行） |

**示例：**
- `awk '{print $1}'` - 打印第一列
- `awk -F: '{print $1}' /etc/passwd` - 以冒号分隔
- `awk 'NR==10' file.txt` - 打印第10行
- `awk '{sum+=$1} END {print sum}'` - 计算第一列总和
- `awk 'length>80' file.txt` - 打印超过80字符的行

### head/tail - 查看文件开头/结尾
| 选项 | 含义 |
|------|------|
| -n N | 显示 N 行 |
| -c N | 显示 N 字节 |
| -f | 持续追踪文件变化（tail） |
| -F | 同 -f，但文件被删除重建后继续追踪 |

**示例：**
- `head -20 file.txt` - 前20行
- `tail -f log.txt` - 实时查看日志
- `head -n 100 file.txt | tail -n 20` - 第81-100行

### wc - 统计
| 选项 | 含义 |
|------|------|
| -l | 统计行数 |
| -w | 统计单词数 |
| -c | 统计字节数 |
| -m | 统计字符数 |
| -L | 最长行的长度 |

### sort - 排序
| 选项 | 含义 |
|------|------|
| -n | 按数字排序 |
| -r | 反向排序 |
| -k N | 按第 N 列排序 |
| -t CHAR | 指定分隔符 |
| -u | 去重 |
| -h | 人类可读数字排序（K, M, G） |

**示例：**
- `sort file.txt` - 按字母排序
- `sort -n -k2 file.txt` - 按第2列数字排序
- `sort -t: -k3 -n /etc/passwd` - 按UID排序

### uniq - 去重
| 选项 | 含义 |
|------|------|
| -c | 显示每行出现的次数 |
| -d | 只显示重复的行 |
| -u | 只显示不重复的行 |
| -i | 忽略大小写 |

⚠️ **注意**：uniq 只能去除相邻的重复行，通常需要先 sort

**示例：**
- `sort file.txt | uniq` - 排序后去重
- `sort file.txt | uniq -c | sort -rn` - 统计每行出现次数并排序

### cut - 提取列
| 选项 | 含义 |
|------|------|
| -d CHAR | 指定分隔符 |
| -f N | 提取第 N 列（可用逗号或范围） |
| -c N | 提取第 N 个字符 |
| -s | 不显示不含分隔符的行 |

**示例：**
- `cut -d: -f1 /etc/passwd` - 提取第一列（用户名）
- `cut -c1-10 file.txt` - 提取前10个字符
- `cut -d, -f2,4 file.csv` - 提取第2和第4列

### tr - 字符转换
| 选项 | 含义 |
|------|------|
| -d | 删除指定字符 |
| -s | 压缩重复字符 |
| -c | 取补集 |

**示例：**
- `tr 'a-z' 'A-Z'` - 转大写
- `tr -d '\n'` - 删除换行符
- `tr -s ' '` - 压缩多个空格为一个
- `tr -cd 'a-zA-Z0-9'` - 只保留字母和数字

## 系统与进程

### ps - 进程状态
| 选项 | 含义 |
|------|------|
| -e 或 -A | 显示所有进程 |
| -f | 完整格式 |
| -u USER | 指定用户的进程 |
| aux | BSD 风格的常用组合（显示所有进程详情） |
| --forest | 以树形显示进程关系 |

### kill - 发送信号
| 信号 | 名称 | 含义 |
|------|------|------|
| 1 | SIGHUP | 挂起/重载配置 |
| 2 | SIGINT | 中断（同 Ctrl+C） |
| 9 | SIGKILL | 强制终止（不可捕获） |
| 15 | SIGTERM | 正常终止（默认） |
| 18 | SIGCONT | 继续运行 |
| 19 | SIGSTOP | 暂停运行 |

**示例：**
- `kill 1234` - 发送 SIGTERM
- `kill -9 1234` - 强制终止
- `kill -l` - 列出所有信号

### top/htop - 进程监控
实时显示系统状态和进程信息。

**top 交互命令：**
- `q` - 退出
- `k` - 杀进程
- `M` - 按内存排序
- `P` - 按 CPU 排序
- `1` - 显示每个 CPU 核心

### df - 磁盘空间
| 选项 | 含义 |
|------|------|
| -h | 人类可读格式 |
| -T | 显示文件系统类型 |
| -i | 显示 inode 信息 |

### du - 目录大小
| 选项 | 含义 |
|------|------|
| -h | 人类可读格式 |
| -s | 只显示总计 |
| -d N | 显示 N 层深度的子目录 |
| --max-depth=N | 同 -d N |

**示例：**
- `du -sh *` - 显示当前目录下各项大小
- `du -h --max-depth=1` - 显示一层子目录大小

## 权限管理

### chmod - 修改权限
| 数字 | 权限 |
|------|------|
| 4 | 读 (r) |
| 2 | 写 (w) |
| 1 | 执行 (x) |
| 7 | rwx (4+2+1) |
| 6 | rw- (4+2) |
| 5 | r-x (4+1) |

**符号模式：**
- `u` = 用户, `g` = 组, `o` = 其他, `a` = 所有
- `+` = 添加, `-` = 移除, `=` = 设置

**示例：**
- `chmod 755 script.sh` - rwxr-xr-x
- `chmod +x script.sh` - 添加执行权限
- `chmod u+w,g-w file` - 用户加写，组减写
- `chmod -R 644 dir/` - 递归修改

### chown - 修改所有者
| 选项 | 含义 |
|------|------|
| -R | 递归修改 |
| -v | 显示过程 |

**示例：**
- `chown user file` - 修改所有者
- `chown user:group file` - 修改所有者和组
- `chown :group file` - 只修改组
- `chown -R user:group dir/` - 递归修改

### sudo - 超级用户执行
| 选项 | 含义 |
|------|------|
| -u USER | 以指定用户执行 |
| -i | 模拟登录 shell（加载环境） |
| -s | 以 root shell 执行 |
| -l | 列出可执行的命令 |
| -k | 清除密码缓存 |

## 网络相关

### curl - 数据传输
| 选项 | 含义 |
|------|------|
| -O | 使用远程文件名保存 |
| -o FILE | 指定输出文件名 |
| -I | 只获取响应头 |
| -i | 显示响应头和内容 |
| -X METHOD | 指定 HTTP 方法（GET, POST, PUT, DELETE） |
| -H "Header" | 添加请求头 |
| -d "data" | 发送 POST 数据 |
| -F "file=@path" | 上传文件 |
| -L | 跟随重定向 |
| -u user:pass | 基本认证 |
| -s | 静默模式 |
| -v | 显示详细信息 |

**示例：**
- `curl https://example.com` - 获取网页
- `curl -O https://example.com/file.zip` - 下载文件
- `curl -X POST -d '{"key":"val"}' URL` - POST JSON

### wget - 下载文件
| 选项 | 含义 |
|------|------|
| -c | 断点续传 |
| -r | 递归下载 |
| -np | 不追溯到父目录 |
| -nd | 不创建目录结构 |
| -O FILE | 指定输出文件名 |
| -b | 后台下载 |
| -q | 静默模式 |

### ssh - 远程连接
| 选项 | 含义 |
|------|------|
| -p PORT | 指定端口 |
| -i KEY | 指定密钥文件 |
| -L LOCAL:HOST:REMOTE | 本地端口转发 |
| -R REMOTE:HOST:LOCAL | 远程端口转发 |
| -N | 不执行远程命令（只做端口转发） |
| -T | 不分配终端 |

**示例：**
- `ssh user@host` - 连接远程主机
- `ssh -p 2222 user@host` - 指定端口
- `ssh -i ~/.ssh/key.pem user@host` - 使用密钥

### scp - 远程复制
| 选项 | 含义 |
|------|------|
| -r | 递归复制目录 |
| -P PORT | 指定端口 |
| -p | 保留文件属性 |
| -v | 显示详细信息 |

**示例：**
- `scp file user@host:/path` - 本地到远程
- `scp user@host:/path/file .` - 远程到本地
- `scp -r dir user@host:/path` - 复制目录

### ping - 测试网络连通性
| 选项 | 含义 |
|------|------|
| -c N | 发送 N 个包后停止 |
| -i N | 发送间隔（秒） |
| -t N | 设置 TTL |
| -s N | 设置包大小 |

## 压缩与解压

### tar - 打包/解压
| 选项 | 含义 |
|------|------|
| -c | 创建归档 |
| -x | 解压归档 |
| -t | 列出内容 |
| -v | 显示过程 |
| -f FILE | 指定文件名 |
| -z | 使用 gzip |
| -j | 使用 bzip2 |
| -J | 使用 xz |
| -C DIR | 解压到指定目录 |

**示例：**
- `tar -cvf archive.tar files` - 创建 tar 包
- `tar -czvf archive.tar.gz files` - 创建 gzip 压缩包
- `tar -xvf archive.tar` - 解压 tar 包
- `tar -xzvf archive.tar.gz` - 解压 gzip 包
- `tar -xzvf archive.tar.gz -C /target` - 解压到指定目录

### zip/unzip
| 选项 | 含义 |
|------|------|
| zip -r | 递归压缩目录 |
| unzip -l | 列出内容 |
| unzip -d DIR | 解压到指定目录 |

### gzip/gunzip
| 选项 | 含义 |
|------|------|
| -d | 解压 |
| -k | 保留原文件 |
| -v | 显示信息 |
| -N | 压缩级别（1-9） |

**示例：**
- `gzip file` - 压缩（原文件被删除）
- `gzip -k file` - 压缩并保留原文件
- `gunzip file.gz` - 解压
